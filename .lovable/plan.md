

# seQRets Landing Page

A dark-themed, full marketing landing page for seQRets — inspired by the dramatic crypto-native aesthetic of Cypherock, using the seQRets brand color palette (#080707 dark background, #e8e5df / #d3cdc1 warm accents, #3e3739 mid-tones).

## Sections (top to bottom)

### 1. Navigation Bar
- seQRets logo (using the provided light theme logo asset on dark background) + "Secure. Split. Share." tagline
- Nav links: Features, How It Works, Security, Desktop App
- "Get Started Free" CTA button linking to seqrets.app

### 2. Hero Section
- Bold headline emphasizing the core value proposition: "Your Secrets, Split Into Unbreakable Pieces"
- Subtitle explaining Shamir's Secret Sharing in plain language
- Two CTAs: "Try the Free Web App →" and "Get the Desktop App" (coming soon badge)
- Subtle animated background or gradient effect for visual impact

### 3. How It Works
- Visual 3-step flow (Encrypt → Split → Distribute) with icons/illustrations
- Clean, easy-to-understand explanation of the Qard concept
- Emphasize that everything runs locally — secrets never leave your device

### 4. Features Grid
- Core features with icons: Secure secrets, Restore from QR codes, Inheritance planning, Smart card support, BIP-39 optimization, AI assistant (Bob)
- Highlight web vs desktop differences subtly

### 5. Security Architecture
- Visually compelling section showcasing the crypto stack (Argon2id, XChaCha20-Poly1305, Shamir's Secret Sharing)
- Encrypt-first architecture diagram
- Quantum resistance callout
- "Zero knowledge — your secrets never leave your device" emphasis

### 6. Web vs Desktop Comparison Table
- Side-by-side comparison (free web app vs official signed desktop release)
- Features like auto-updates, code signing, smart card, portable reader
- Desktop column highlights what's included in the purchase

### 7. Desktop App Purchase Section (Coming Soon)
- Prominent "Coming Soon" banner with email signup / waitlist form
- What you'll get: signed binary, auto-updates, smart card, USB reader
- "Why pay when it's open source?" explainer
- Build-it-yourself option with GitHub link

### 8. Open Source & Trust
- AGPLv3 badge, GitHub link
- Emphasize transparency and auditability
- Community/contributing callout

### 9. Footer
- Links: GitHub, Web App, Documentation
- © 2026 seQRets. All rights reserved.
- "Your security is your responsibility. Use with caution."

## Design Details
- Dark background (#080707) with warm beige/cream accents (#e8e5df, #d3cdc1)
- Bold typography similar to the app screenshots
- Subtle card elements with #3e3739 backgrounds
- Smooth scroll animations between sections
- Fully responsive (mobile-friendly)
- The provided logo images will be embedded as assets

## No Backend Needed
This is a purely static landing page. The email waitlist can be a simple mailto link or a placeholder form (no backend required unless you want to collect emails, which we can add later with Supabase).
